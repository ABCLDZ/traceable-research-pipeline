# Argument-governance experiment

该目录保存重型Source—Evidence—Claim—Inference—ChangeSet治理实验的终止结果。

终止性试验中，严格模式的影响召回率为76.19%，强普通流程为95.24%；
严格模式留下3个P1级下游漏改。项目据此停止扩张重型语义治理。

这里的文件仅作为评估与产品决策档案，不是生产运行时依赖。生产主线只回收：

- EvidenceCard到EvidenceRecord的准入边界；
- ResearchBrief中的范围、反证和开放问题；
- 发布版本与哈希冻结。

完整冻结链包含未公开的本地试验材料，因此不进入公共仓库。
本目录只携带终评报告和最终指标，不伪装成可独立复验的完整旧实验副本。
